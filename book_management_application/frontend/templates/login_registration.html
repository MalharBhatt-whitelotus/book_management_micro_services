Book Management System

Inventory, checkout, and billing in one clean workflow.

A complete book store and inventory platform built with FastAPI, SQLAlchemy, SQLite, JWT authentication, and role-based dashboards for both admin and user workflows.

Admin Inventory Control

Add, update, delete books, monitor stock, and review billing history.

User Purchase Flow

Browse available books, select quantities, checkout, and receive generated bills.

Automatic Stock Updates

Every successful checkout reduces inventory and stores the bill in the database.

Tip: The default admin credentials come from your .env file.

Welcome back

Login to continue, or create a new user account to start browsing and purchasing books.

Admin login: use the admin username/password from your .env file.
User login: register a normal user account from the Register tab.
After successful registration, you can login immediately using your new credentials.
